概要

Collapsible Columnsプラグインは、colspanプロパティが定義されたヘッダーで覆われた列を折りたたむことを可能にします。
このプラグインは、（Nested Headersプラグインで作成された）「展開/折りたたみ」ボタンの複数列ヘッダーを追加します。 これらのボタンをクリックすると、すべての「子」ヘッダーが折りたたまれ（展開され）、最初のヘッダーが表示されたままになります。

注:このプラグインが正しく動作するためにはNested Headersプラグインと Hidden Columnsプラグインを有効にする必要があります。

セットアップと設定

Collapsible Columnsプラグインを有効にするには、collapsibleColumnsプロパティを次のいずれかに設定します。

  • true - すべての複数列ヘッダーの機能が有効になります。（colspan属性が定義されているすべての列は[展開/折りたたみ]ボタンで拡張されます）
  • どのヘッダーに「展開/折りたたみ」ボタンを表示するかという情報を含むオブジェクトの配列。この場合のオブジェクトの構造は次のようになります。 
    collapsibleColumns: [
  {row: -4, col: 1, collapsible: true}, // 1列目の4段目（最初の表の行から数えて）のヘッダーにボタンを追加します。
  {row: -3, col: 5, collapsible: true} // 5列目の3段目（最初の表の行から数えて）のヘッダーにボタンを追加します。
]
