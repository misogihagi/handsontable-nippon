Handsontableをインストールするには、次の手順に従います。

  1. インストール
  2. 作成
  3. 初期化
  4. 代替インストール

ステップ1：インストール

  • Handsontable Proをインストールする
  • Community Editionをインストールする

Handsontable Proをインストールする方法はたくさんありますが、npmを使うことをお勧めします。 次のコマンドを入力するだけです。 

npm install handsontable-pro

インストールプロセスが完了したら、このコードをHTMLファイルに埋め込んでください。

<script src="node_modules/handsontable-pro/dist/handsontable.full.min.js"></script>
<link href="node_modules/handsontable-pro/dist/handsontable.full.min.css" rel="stylesheet" media="screen">

あるいは、CDNを使用してください。

<script src="https://cdn.jsdelivr.net/npm/handsontable-pro@/dist/handsontable.full.min.js"></script><link href="https://cdn.jsdelivr.net/npm/handsontable-pro@/dist/handsontable.full.min.css" rel="stylesheet" media="screen">

Handsontable CEをインストールする方法はたくさんありますが、npmを使うことをお勧めします。 次のコマンドを入力するだけです。 

npm install handsontable

インストールプロセスが完了したら、このコードをHTMLファイルに埋め込んでください。

<script src="node_modules/handsontable/dist/handsontable.full.min.js"></script>
<link href="node_modules/handsontable/dist/handsontable.full.min.css" rel="stylesheet" media="screen">

あるいは、CDNを使用してください。

<script src="https://cdn.jsdelivr.net/npm/handsontable@/dist/handsontable.full.min.js"></script><link href="https://cdn.jsdelivr.net/npm/handsontable@/dist/handsontable.full.min.css" rel="stylesheet" media="screen">

ステップ2：作成

スプレッドシートに変換される空の<div>要素を追加します。この要素に "example" IDを付けましょう。

<div id="example"></div>

ステップ3：初期化

次のステップでは、その<div id="example">要素への参照を渡し、それにサンプルデータを入力します。

var data = [
  ["", "Ford", "Tesla", "Toyota", "Honda"],
  ["2017", 10, 11, 12, 13],
  ["2018", 20, 11, 14, 13],
  ["2019", 30, 15, 12, 13]
];

var container = document.getElementById('example');
var hot = new Handsontable(container, {
  data: data,
  rowHeaders: true,
  colHeaders: true,
  filters: true,
  dropdownMenu: true
});

ステップ4：結果

これで、Handsontable Proが起動して使用可能になりました。

データソースをバインドするだけでなく、 Handsontable Proで行われた変更を保存する方法を疑問に思っているでしょうか。詳細については、バインディングデータを参照してください。
