以下に、Handsontable ProおよびCommunity Editionの機能の最も完全なリストを示します:

Proの特徴

名称

説明 Binding rows with headers
行番号をロックし、常に行をセルの横に表示するために、行をヘッダーでバインドします。 Collapsing columns
内容に合わせて列を展開したり折りたたんだりします。 Column summary
定義済みの計算を行い、その結果をHandsontableに表示します。 Dropdown menu
列に追加の操作を有効にするために、見出しにメニューを追加します。 Export to file
CSVや文字列などのフラットファイルにデータをエクスポートします。 Filtering
条件を満たす行を表示し、残りを隠します。 Header tooltips
ツールチップにヘッダーラベルを表示します。 Hiding columns
特定の列を非表示にして残りを表示します。 Hiding rows
特定の行を非表示にして残りを表示します。 Multiple column sorting
複数の列にわたってデータを昇順または降順に並べ替えます。 Nested headers
データをグループにまとめるための、ネストした階層構造のヘッダを作成します。 Trimming rows
特定の行を、表示されないようにレンダリングプロセスから除外します。 Experimental Proの特徴（現段階でサポートはなし）

名称

説明 Alignment
内容がセルまたはセルの範囲のどこに置かれるかを決定します。 Autocomplete
入力する際、提案されたオプションのうち1つを選びます。 Checkbox
真か偽かを入力するためのチェックボックスをセルに追加します。 Comments
セルの内容の理解の手助けとして、追加の入力欄を提供します。 Conditional formatting
セルの値に応じてのセルの書式をどうするかを定義する。条件付き書式。 Context menu
ショートカットメニューを呼び出して、選択したオブジェクトに関連する操作を選択します。 Custom borders
セルまたはセルの範囲の境界線に任意のスタイルを適用します。 Custom buttons
カスタムアクションボタンを使用して列または行を挿入および削除します。 Custom HTML
HTMLコンテンツをセルまたはヘッダーに表示します。 Data validation
セルに入力できる値を制御します。 Date
日付入力のカレンダーを使用して日付を選択します。 Disable cell editing
外観と動作を変更せずにセルを編集不可能にします。 Auto fill
フィルハンドル（セルの右下隅にある正方形）をドラッグして、系列的にデータをセルに入力します。 Dropdown
セル内の選択肢の詳細なドロップダウンリストを作成します。 Fixed rows and columns
テーブルを移動する間表示される行または列を定義します。 Freezing
スクロール中に列を表示したままにするために固定列を作成します。 Handsontable
カスタムセルエディタとしてHandsontable自体を使用します。 Highlighting current
アクティブな列または行全体を示します。 Internationalization
特定の言語に翻訳するためにUIのテキストを変更します。 Merge cells
複数の行または列にわたるセルを表示します。 Moving
行または列をドラッグしてテーブル内で入れ替えます。 Numeric
セルを数値として理解します。 Password
セル内の値をアスタリスクで伏せて表示します。 Pre-populating new rows
空のセルを事前定義されたタイプや値で作成します。 Read-only
変更しないようにセルまたはセル範囲をロックします。 Resizing
サイジングハンドルをドラッグして、列または行のサイズを変更します。 Scrolling
テーブル内を移動するために、ブラウザ由来のスクロールバーを使用します。 Search for values
検索フィールドで内容を検索します。 Select
セル内に簡単なドロップダウンリストを作成します。 Sorting data
列全体でデータを昇順または降順に並べ替えます。 Stretching
