Commentsプラグインを使用すると、Handsontableでコメントを簡単に追加、編集、削除できます。

プラグインを有効にする

プラグインを使用するには、commentsプロパティをtrueに設定する必要があります。 プラグインを有効にして、必要なすべてのコンテキストメニュー項目を追加します。 例えば：

let hot = new Handsontable(container, {
  data: Handsontable.helper.createSpreadsheetData(10, 10),
  comments: true
});

コンテキストメニューからコメントを追加する

プラグインを有効にすると、コンテキストメニューに新しい項目が追加されます。

  • コメントを追加・編集
  • コメントを削除
  • 読み取り専用コメント

私はそれらのすべてがかなり見ればわかると思います。

プリセットコメントを設定する

テーブルにコメントを事前定義することもできます。コメントはテーブル/列/セルのメタデータオブジェクトに格納されているので、 その型の他のプロパティとして宣言できます。
例：

cell: [
  {row: 1, col: 1, comment: {value: 'Hello world!'}}
]

この例では（1,1）のセルに"Hello world！"コメントを追加しています。

API

APIを使用して、コメント関連の情報のほとんどを追加、削除、および修正することができます。詳しくはコメントを参照してください。
