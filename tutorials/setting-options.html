セルオプションの紹介

Handsontableコンストラクターに渡されたcell配列を使用して、任意のコンストラクターまたは列のオプションを特定のセル（行と列の組み合わせ）に対して上書きすることができます。例：

var hot = new Handsontable(document.getElementById('example'), {
  cell: [
    {row: 0, col: 0, readOnly: true}
  ]
});

またはHandsontableコンストラクターにセル関数プロパティを使用する。例：

var hot = new Handsontable(document.getElementById('example'), {
  cells: function (row, col, prop) {
    var cellProperties = {}

    if (row === 0 && col === 0) {
      cellProperties.readOnly = true;
    }

    return cellProperties;
  }
})

階層設定とは?

Handsontable 0.9以降、階層設定を使用しています。 これは、テーブル全体の設定オプションとその列および特定のセルをすばやく提供する方法です。

次の例を見てください。

var hot = new Handsontable(document.getElementById('example'), {
  readOnly: true,
  columns: [
    {readOnly: false},
    {},
    {}
  ],
  cells: function (row, col, prop) {
    var cellProperties = {}

    if (row === 0 && col === 0) {
      cellProperties.readOnly = true;
    }

    return cellProperties;
  }
});

上記の書き方では、最初の列のTDを除いて、すべてのTDが読み取り専用になります。 一方左上隅のTDを除いて最初の列のTDは編集可能で、左上隅のTDは読み取り専用になります。

階層設定モデル

階層設定モデルはプロトタイプの継承に基づいています。 これは、jQueryの拡張を使用した以前のモデルと比べてはるかに高速でメモリ効率がよくなります。それを自分で見てください： http://jsperf.com/extending-settings

  • コンストラクター

    第1レベルを使用して提供される設定オプション

    new Handsontable(document.getElementById('example'), {
  option: 'value'
});

    とupdateSettingsメソッド

  • 列

    第2レベルのオブジェクトを使用して提供される設定オプション

    new Handsontable(document.getElementById('example'), {
  columns: {
    option: 'value'
  }
});

  • セル

    第2レベルを使用して提供される設定オプション

    new Handsontable(document.getElementById('example'), {
  cells: function(row, col, prop) {

  }
});
