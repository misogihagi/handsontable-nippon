カスタム枠線機能を有効にするには、customBordersオプションを設定します。 trueに設定することも、事前定義された設定で配列として初期化することもできます。

Handsontableを定義済みのカスタムな枠線で初期化するには、セル座標とボーダースタイルを配列の形式で指定します。

  • 行と列を指定する: {row: 2, col: 2, left: { /*...*/ }}
  • 範囲を指定する: {range: {from: {row: 1, col: 1}, to:{row: 3, col: 4}}, left: { /*...*/ }}
