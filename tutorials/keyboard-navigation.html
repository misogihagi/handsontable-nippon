移動

  • Arrow Up ↑ - 現在アクティブなセルの上にあるセルに移動します（存在する場合）
  • Arrow Down ↓ - 現在アクティブなセルの下にあるセルに移動します（存在する場合）
  • Arrow Right → - 現在のアクティブセルの右にあるセルに移動します（存在する場合）
  • Arrow Left ← - 現在アクティブなセルの左にあるセルに移動します（存在する場合）
  • Tab - 現在アクティブなセルの右にあるセルに移動します（存在する場合）
  • Tab + Shift - 現在アクティブなセルの左にあるセルに移動します（存在する場合）
  • Home - 行の最初のセルに移動します
  • End - 行の最後のセルに移動します
  • Ctrl + Home - 列の最初のセルに移動します
  • Ctrl + End - 列の最後のセルに移動します

選択

  • Ctrl + A - すべて選択
  • Shift + Arrow Up ↑ - 上にセルの選択範囲を広げます
  • Shift + Arrow Down ↓ - 下のセルの選択範囲を広げます
  • Shift + Arrow Right → - 右側のセルの選択範囲を広げます
  • Shift + Arrow Left ← - 左側のセルの選択範囲を広げます
  • Shift + Home - 現在のセルを含む、左側の行にあるすべてのセルを選択します
  • Shift + End - 現在のセルを含む、右側の行にあるすべてのセルを選択します
  • Ctrl + Shift + Home - 現在のセルを含めて、列の一番上にあるセルをすべて選択します
  • Ctrl + Shift + End - 現在のセルを含めて、列の一番下にあるセルをすべて選択します

編集

  • Enter - セルエディターを開く/閉じます
  • F2 - セルエディターを開きます
  • Esc - 編集をキャンセルしてセルエディタを閉じます
  • Backspace - セルを空にします
  • Delete - セルを空にします
  • Ctrl + C - セルの内容をコピーします
  • Ctrl + X - セルの内容を切り取ります
  • Ctrl + V - セルの内容を貼り付けます
  • Ctrl + Enter - 選択したすべてのセルに編集したセルの値を入力します
  • Ctrl + Z - 元に戻します
  • Ctrl + Y - やり直します

コンテキストメニュー

  • Arrow Down ↓ - コンテキストメニューの次のオプションに移動します
  • Arrow Up ↑ - コンテキストメニューの前のオプションに移動します
  • Enter - コンテキストメニューからオプションを選択します

セルの結合

  • Ctrl + M - 選択したセルを結合/結合解除します
