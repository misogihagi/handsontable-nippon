プラグインには、Handsontableに簡単に差し込むことができる1つ以上の機能が含まれています。 新しいプラグインを書くのは難しい作業ではありません。 Skeleton template テンプレートを複製するだけでよい始まりとなります。

プラグインには、内部と外部の2種類があります。 どちらもHandsontableの機能を拡張していますが、前者はHandsontableビルドに組み込まれており、 後者は別のファイルからインクルードする必要があります。 プラグインについてさらに学ぶにはSkeletonガイドを参照してください。

個人のニーズのためのプラグインを作成することは、通常、他の開発者が使用することを目的としたアプリケーションを書くよりも簡単です。 そのようなコードは適切に記述され、文書化され、頻繁に更新されそしてサポートされなければなりません。 もしあなたが他の開発者と使用することを念頭にプラグインを作成するなら、私たちと共有することを検討してください。 詳細につきまして連絡を参照してください。

最後になりましたが、v1.11.0-beta1以降、私たちはECMAScript 6をサポートし、コードスタイル（ Airbnb JavaScript Styleがもとになっている）に従うので、新しいプラグインにはこれらの標準に従っていることが必要です。
